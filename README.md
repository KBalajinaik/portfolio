# Full Stack Web Developer Portfolio Website

A modern, premium, fully responsive, glassmorphism-themed portfolio website built to showcase skills, experience, projects, and contact info.

## Features
- **Design**: Premium Glassmorphism styling with dark-mode default and toggle support.
- **Dynamic Elements**: Tagline typing animation, responsive carousel for reviews, and animated progress bars.
- **Projects Showcase**: Interactive categorization filtering and instant live search matching titles or tools.
- **Resume Portal**: Container Query-driven resume mockup sheet that scales to fit any display device size.
- **Form Verification**: Local checking logic for input validation with custom toast notification outputs.

## Technology Stack
- **Structure**: Semantic HTML5
- **Styling**: Vanilla CSS3
- **Logic**: Vanilla ES6+ Javascript
- **Icons**: Lucide Icons CDN integration
- **Assets**: Professional AI-generated PNG image profiles/mockups

## Local Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/KBalajinaik/portfolio.git
   ```
2. Open `index.html` in any contemporary web browser to view the portfolio.
